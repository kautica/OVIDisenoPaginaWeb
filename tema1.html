TEMA 1: Controlador de versiones GITHUB

Objetivo del ovi

Realizar un OVI a través de tecnologías basadas en HTML y CSS cuyo fin es dar a conocer acerca del uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para la creación de contenidos en formato web.

  • Objetivo específico 1:
    Administrar y operar una cuenta GITHUB
  • Objetivo específico 2:
    Crear un guion para la realización de un OVI con su respectivo mockup o wireframe.
  • Objetivo específico 3:
    Desarrollar el HTML5 y CSS3 que contenga el desarrollo del OVI, según el tema elegido

Objetivo del tema

Dar a conocer la temática de GIT y GITHUB para que el estudiante logre administrar y operar un sistema controlador de versiones como lo es GIT o GITHUB
